# Taxi User Behavior Analysis

## 📌 Project Overview

This project investigates user behavior in a taxi service platform.

The objective is to analyze trip distance distributions for different taxi classes and test the hypothesis that users prefer Comfort class for longer trips.

---

## 🎯 Business Question

Do users choose Comfort class over Economy for long-distance trips?

---

## 📊 Analytical Tasks

- Explore trip distance distributions
- Compare Economy and Comfort classes
- Analyze descriptive statistics
- Visualize density distributions
- Test statistical hypothesis

---

## ⚙️ Tools

- Python
- Pandas
- Matplotlib / Seaborn
- SciPy (hypothesis testing)
