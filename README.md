# Анализ поведения пользователей сервиса такси

## Обзор проекта

Данный проект анализирует поведение пользователей на платформе сервиса такси, с фокусом на предпочтениях по дистанции поездки в разных классах обслуживания.

Цель — исследовать, выбирают ли пользователи класс Comfort вместо Economy для поездок на большие расстояния.

---

## Бизнес-вопрос

Предпочитают ли пользователи класс Comfort вместо Economy для дальних поездок?

---

## Гипотеза

- **H0 (Нулевая гипотеза):** Различий в распределении дистанций поездок между классами Economy и Comfort нет.
- **H1 (Альтернативная гипотеза):** Поездки класса Comfort имеют большую дистанцию по сравнению с Economy.

---

## Методология

1. Разведочный анализ данных (EDA)
   - Анализ структуры набора данных
   - Описательная статистика
   - Анализ распределения по классам

2. Визуализация распределений
   - Kernel Density Estimation (KDE)
   - Сравнение boxplot

3. Проверка нормальности
   - Тест Шапиро–Уилка (с использованием выборочного подхода)

4. Статистическое тестирование
   - Тест Манна–Уитни (непараметрический)

---

## Ключевые результаты

### Анализ распределений

Распределения дистанций для классов Economy и Comfort не являются нормальными.  
Гистограммы и boxplot-графики демонстрируют правостороннюю асимметрию и наличие длинного хвоста, что делает применение параметрических тестов некорректным.

### Поведенческие различия по дистанции

Поездки класса Comfort демонстрируют более высокую концентрацию больших дистанций по сравнению с Economy.

Медианные значения и верхние квартили для Comfort выше, что подтверждает смещение распределения в сторону длинных поездок.

### Статистическая значимость

Непараметрический тест Манна–Уитни показал статистически значимое различие между распределениями дистанций:

p-value < 0.05

Нулевая гипотеза о равенстве распределений отвергается.

### Поведенческая интерпретация

Дистанция поездки статистически связана с выбором класса такси.

Пользователи, совершающие более длинные поездки, чаще выбирают Comfort, что может объясняться:

- Снижением ценовой чувствительности при увеличении дистанции  
- Более высокой ценностью комфорта на длительных маршрутах  
- Сегментацией спроса по длине поездки  

### Выводы

Результаты позволяют:

- Подтвердить наличие поведенческой сегментации пользователей  
- Использовать дистанцию как фактор динамического ценообразования  
- Продвигать класс Comfort для длинных маршрутов  
- Применять дистанцию как предиктор в моделях выбора тарифа  

Дистанция поездки является значимой переменной для оптимизации выручки и продуктовой стратегии.

## Ограничения

- В анализе не учитываются временные факторы (сезонность).
- Цель поездки неизвестна (бизнес или личная).
- Одна лишь дистанция может не полностью объяснять выбор класса.
- Структура набора данных может не отражать всю совокупность пользователей.
