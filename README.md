# Анализ поведения пользователей сервиса такси

## Обзор проекта

Данный проект анализирует поведение пользователей на платформе сервиса такси, с фокусом на предпочтениях по дистанции поездки в разных классах обслуживания.

Цель — исследовать, выбирают ли пользователи класс Comfort вместо Economy для поездок на большие расстояния.

---

## Бизнес-вопрос

Предпочитают ли пользователи класс Comfort вместо Economy для дальних поездок?

---

## Гипотеза

- **H0 (Нулевая гипотеза):** Различий в распределении дистанций поездок между классами Economy и Comfort нет.
- **H1 (Альтернативная гипотеза):** Поездки класса Comfort имеют большую дистанцию по сравнению с Economy.

---

## Методология

1. Разведочный анализ данных (EDA)
   - Анализ структуры набора данных
   - Описательная статистика
   - Анализ распределения по классам

2. Визуализация распределений
   - Kernel Density Estimation (KDE)
   - Сравнение boxplot

3. Проверка нормальности
   - Тест Шапиро–Уилка (с использованием выборочного подхода)

4. Статистическое тестирование
   - Тест Манна–Уитни (непараметрический)

---

## Ключевые инсайты

### Анализ распределений

Распределения дистанций поездок классов Economy и Comfort являются асимметричными и правосторонне скошенными, что делает применение параметрических тестов некорректным.

Для проверки гипотезы использован непараметрический тест Манна–Уитни.


### Сравнение медианных значений

- Медиана Economy: **23.80 км**
- Медиана Comfort: **28.08 км**
- Абсолютная разница: **+4.28 км**
- Относительное увеличение: **+17.98%**

Таким образом, типичная поездка Comfort почти на 18% длиннее типичной поездки Economy.


### Сравнение средних значений

- Среднее Economy: **26.28 км**
- Среднее Comfort: **29.76 км**
- Разница средних: **+3.48 км**
- Относительное увеличение: **+13.22%**

Смещение сохраняется как в медианах, так и в средних значениях, что указывает на устойчивость эффекта.


### Результаты теста Манна–Уитни

- U-статистика: **139702**
- p-value: **0.000642**

p-value < 0.05, что позволяет отвергнуть нулевую гипотезу.

Распределение дистанций Comfort статистически значимо смещено вправо относительно Economy.


### Эффект размера (Probability of Superiority)

Вероятность того, что случайная поездка Comfort длиннее случайной поездки Economy:

**0.559**

Это означает, что в **55.9% случаев** поездка Comfort длиннее, чем случайная поездка Economy.

Это не просто статистическая значимость, а реальный поведенческий сдвиг.


### Проверка устойчивости результата

После удаления верхнего 1% экстремальных дистанций:

- p-value: **0.000521**
- Probability of Superiority: **0.560**

Эффект сохраняется.

Это подтверждает, что различие не вызвано редкими аномально длинными поездками.


## Поведенческая интерпретация

1. Пользователи систематически выбирают Comfort для более дальних поездок.
2. Разница в 4.28 км по медиане является устойчивой и экономически значимой.
3. Вероятность превосходства выше 0.5 подтверждает смещение распределения, а не случайный шум.
4. Эффект устойчив к удалению выбросов.


## Выводы

Полученные результаты позволяют сделать следующие выводы:

- Дистанция является значимым фактором выбора класса такси.
- При увеличении дистанции пользователи становятся менее чувствительными к цене.
- Comfort можно активно продвигать для длинных маршрутов.
- Дистанция может использоваться как предиктор в модели выбора тарифа.
- Возможна реализация динамического апсейла Comfort для поездок выше медианной дистанции Economy.

Исследование выявляет статистически значимую поведенческую закономерность, однако не доказывает причинно-следственную связь.

Пользователи чаще выбирают класс Comfort для дальних поездок.

Компания может:

- Оптимизировать ценовую стратегию для длительных поездок
- Персонализировать рекомендации на основе паттернов дистанции поездок

## Ограничения

- В анализе не учитываются временные факторы (сезонность).
- Цель поездки неизвестна (бизнес или личная).
- Одна лишь дистанция может не полностью объяснять выбор класса.
- Структура набора данных может не отражать всю совокупность пользователей.
